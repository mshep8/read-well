

# READ. — Adult Literacy Learning App

## Overview
A respectful, empowering mobile-first web app that helps adults build reading skills through a scaffolded learning path — from foundational phonics to real-world document comprehension. All progress is stored locally on-device.

---

## Design Direction
- **Warm, sophisticated palette**: Deep navy/slate primary, warm sand/cream backgrounds, muted teal accents — replacing the current black-and-white scheme which feels stark
- **Typography**: Large, clean sans-serif font (18px+ body text), generous line spacing for readability, dyslexia-friendly considerations
- **Tone**: Every piece of copy written for adults — supportive and encouraging without being patronizing
- **Layout**: Spacious, uncluttered screens with clear visual hierarchy. Icons paired with text on all buttons for users who may struggle reading labels alone
- **No stock photos with watermarks** — clean illustrations or simple iconography instead

---

## Screens & Flow

### 1. Welcome / Landing Screen
- Clean "READ." branding with a warm tagline: *"Your reading journey starts here"*
- Two clear actions: **"Get Started"** (new user) and **"Continue Learning"** (returning user)
- Audio icon to hear the welcome message read aloud (placeholder)
- Feels calm and private — no imagery suggesting school or childhood

### 2. Onboarding (New Users — 2-3 steps)
- Simple name entry (first name only, no email/password needed since local-only)
- Quick self-assessment: *"Where would you like to start?"* with options like "Letters & Sounds," "Everyday Words," or "Reading Practice" — described with icons, not just text
- Sets the user's starting point without making them feel tested

### 3. Home Dashboard
- Personalized greeting: *"Welcome back, [Name]"*
- **Current lesson card** — large, prominent, showing where they left off with a "Continue" button
- **Learning path overview** — visual progress through 4 stages shown as a simple path/roadmap (not a game map):
  1. Letters & Sounds
  2. Words You'll Use
  3. Sentences & Paragraphs
  4. Real-World Reading
- Daily streak or session counter for gentle motivation
- Bottom navigation bar: Home, Lessons, Progress, Settings

### 4. Lessons Hub
- Four category cards with icons:
  - **Letters & Sounds** — Learn the alphabet, phonics, and how letters make words
  - **Sight Words** — Practice common words you see every day
  - **Sentences & Stories** — Read short passages about everyday topics
  - **Real-World Reading** — Practice with documents like forms, signs, and instructions
- Each card shows completion progress (e.g., "3 of 10 complete")

### 5. Lesson Screen (Phonics Example)
- Large letter display with audio button (placeholder) to hear the sound
- Example words using that letter/sound, each tappable for pronunciation
- Simple interactive exercise: tap the word that starts with the target sound
- Encouraging feedback: *"That's right!"* or *"Not quite — try again"* with no penalty
- "Next" button to advance, "Replay" to hear again

### 6. Practice Screen (Real-World Documents)
- Display a realistic adult document (lease clause, job application section, medication label)
- Highlighted vocabulary words are tappable — shows definition, phonetic pronunciation, and audio icon
- Microphone button placeholder for future speech practice
- Comprehension check: simple question about what was just read

### 7. Progress Screen
- Visual progress bar for each of the 4 learning categories
- Total lessons completed, current streak, time spent
- Milestone celebrations: *"You've completed 10 lessons — keep going!"* with a subtle badge/icon (not childish emojis)
- Encouraging message that adapts to progress level

### 8. Settings
- Change display name
- Text size adjustment (small / medium / large)
- Reset progress option
- About the app

## Lesson Content (Built-in Starter Content)
- **Phonics**: 5 lessons covering vowel sounds and common consonant blends
- **Sight Words**: 3 sets of 10 high-frequency adult words (e.g., "address," "emergency," "schedule")
- **Sentences**: 3 short passages on adult topics (health, workplace, daily life)
- **Real-World**: 2 document exercises (a simplified form, a medication label)

---

## Key UX Improvements Over Current Design
- Replace dark/black buttons with warmer, higher-contrast design
- Add icons alongside all button text
- Remove confusing navigation (hamburger menu replaced with bottom tab bar)
- Audio placeholder icons on all text content
- Consistent visual language across all screens
- Progress is always visible and encouraging
- No screen requires reading instructions to understand what to do

---

## Accessibility
- Minimum 18px font size throughout
- WCAG AA contrast ratios on all text
- Tappable areas minimum 48px
- Icons paired with every text label
- Audio placeholders on all instructional text
- Simple, predictable navigation pattern
